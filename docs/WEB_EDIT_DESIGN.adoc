== 概述

本文档描述 Sprite DSL Web 编辑器的工程落地细节，涵盖架构设计、技术选型、组件体系、构建配置、数据流、工具系统等。

=== 核心目标

- 以 **DSLProgram (AST)** 为单一数据源（Single Source of Truth），实现**文本编辑器 ↔ 画布**双向实时同步
- 使用 **Vue 3 Options API** 管理 UI 状态，保持代码直观可维护
- 使用 **Vite** 作为构建工具，替代当前 `tsc + build-web.js` 的手动打包方式
- **core 层保持零框架依赖**，纯 TypeScript，CLI 和 Web 共用

=== 文档约定

- `src/core/` — 纯 TS 核心模块，零框架依赖，CLI + Web 共用
- `src/web/` — Vue 3 应用，依赖 Vue + Vite
- `src/cli/` — CLI 入口，纯 Node.js

== 技术选型

[cols="1,2,3,3"]
|===
| 层级 | 选型 | 版本 | 选型理由
| 框架 | Vue 3 Options API | ^3.4 | 用户指定；Options API 在复杂度适中时比 Composition API 更直观；`data/computed/methods/watch` 清晰分离
| 构建工具 | Vite 5 | ^5.4 | Vue 3 生态标准构建工具；HMR 开发体验好；默认使用 Rollup 打包；自动处理 TypeScript（esbuild）
| 代码编辑器 | CodeMirror 6 | ^6.0 | 包体积小（gzip ~40KB）；支持自定义 DSL 语法高亮；无框架绑定；支持虚拟滚动
| 语言 | TypeScript | ^5.5 | 项目已有，保持一致
| 样式 | Scoped CSS（Vue SFC）| - | 无需额外 CSS 框架；项目 UI 复杂度适中
| 状态管理 | Vue 3 `reactive` + `provide/inject` | 内置 | 无需 Pinia/Vuex；编辑器状态以 AST 为核心，单组件树层级浅
| 核心模块 | 纯 TypeScript | - | 保持 CLI 和 Web 共用；core 不引入任何框架依赖
| 包管理 | npm | - | 项目已有
|===

=== 组件通信机制

所有组件通信采用 **provide/inject + 直接写 store** 模式，不引入事件总线：

[source,typescript]
----
// App.vue — 根组件提供 editorStore
import { provide } from 'vue'
import { editorStore } from './stores/editorStore'

export default defineComponent({
  setup() {
    provide('editorStore', editorStore)
  },
  // ...
})

// 任意子组件使用
export default defineComponent({
  inject: ['editorStore'],
  computed: {
    currentTool() {
      return this.editorStore.state.currentTool
    }
  },
  methods: {
    onToolClick(tool: string) {
      this.editorStore.state.currentTool = tool
    }
  }
})
----

使用 `provide/inject` 而非 Vuex/Pinia 的理由：

1. 组件树只有两层（App → 子组件），不需要深层的透传
2. 全局状态只有一份 editorStore，单一职责
3. editorStore 本身就是响应式（reactive），修改自动触发视图更新
4. 减少额外依赖

== editorStore 全局状态设计

=== 状态结构

[source,typescript]
----
// stores/editorStore.ts
import { reactive } from 'vue'

export type ToolType = 'select' | 'rect' | 'line' | 'poly' | 'bitmap' | 'eyedropper'

export interface EditorState {
  // ----- DSL 数据源 -----
  dslCode: string                    // DSL 源代码
  program: DSLProgram | null         // 解析后的 AST（单一数据源）
  errors: DSLError[]                 // 解析错误

  // ----- 工具状态 -----
  currentTool: ToolType
  currentColor: string               // 颜色代码 '1'-'9' | 'a'-'z' | 'A'-'Z'
  lineRadius: number                 // 线条半径 0-5
  scale: number                      // 画布缩放倍率 1/2/4/8/16

  // ----- 选中状态 -----
  selectedCommandIndices: number[]   // 选中的命令索引（支持多选）
  highlightedLineNumbers: number[]   // 代码编辑器高亮的行号

  // ----- 鼠标状态 -----
  mouseDSLPosition: Point | null     // 鼠标在 DSL 坐标中的位置

  // ----- 撤销/重做 -----
  undoStack: string[]                // DSL 代码快照历史
  redoStack: string[]                // 重做栈

  // ----- 工具预览 -----
  toolPreview: any                   // 当前工具的预览数据
}

export function createEditorStore() {
  const state = reactive<EditorState>({
    dslCode: '',
    program: null,
    errors: [],
    currentTool: 'select',
    currentColor: '1',
    lineRadius: 0,
    scale: 4,
    selectedCommandIndices: [],
    highlightedLineNumbers: [],
    mouseDSLPosition: null,
    undoStack: [],
    redoStack: [],
    toolPreview: null,
  })

  // ---------- 核心方法 ----------

  /**
   * 解析 DSL 代码并更新 AST + 渲染
   * 调用链：用户输入 → CodeMirror emit → 此方法 → 画布更新
   */
  function parseAndRender(code: string): void {
    state.dslCode = code

    if (!code.trim()) {
      state.program = null
      state.errors = []
      return
    }

    const { program, errors } = parseDSL(code)
    state.program = program
    state.errors = errors
    // CanvasEditor 通过 watch('program') 自动重绘
  }

  /**
   * 从 AST 生成 DSL 代码
   * 调用链：画布绘制/编辑 → AST 修改 → 此方法 → 代码编辑器更新
   */
  function syncCodeFromAST(): void {
    if (!state.program) return
    const code = generateDSL(state.program)
    state.dslCode = code
    // CodeEditor 通过 watch('dslCode') 自动更新编辑器内容
  }

  /**
   * 添加新命令（画布绘制完成时调用）
   */
  function addCommand(cmd: DrawCommand | UseCmd): void {
    if (!state.program) return
    pushSnapshot()
    state.program.commands.push(cmd)
    syncCodeFromAST()
  }

  /**
   * 修改命令（拖拽/编辑时调用）
   */
  function modifyCommand(index: number, cmd: DrawCommand | UseCmd): void {
    if (!state.program || index < 0 || index >= state.program.commands.length) return
    pushSnapshot()
    state.program.commands[index] = cmd
    syncCodeFromAST()
  }

  /**
   * 删除命令
   */
  function deleteCommand(index: number): void {
    if (!state.program) return
    pushSnapshot()
    state.program.commands.splice(index, 1)
    syncCodeFromAST()
  }

  // ---------- 撤销/重做 ----------

  function pushSnapshot(): void {
    const current = state.dslCode
    if (state.undoStack.length > 0 && state.undoStack[state.undoStack.length - 1] === current) return
    state.undoStack.push(current)
    if (state.undoStack.length > 100) state.undoStack.shift()
    state.redoStack = []
  }

  function undo(): void {
    if (state.undoStack.length === 0) return
    state.redoStack.push(state.dslCode)
    const prev = state.undoStack.pop()!
    parseAndRender(prev)
  }

  function redo(): void {
    if (state.redoStack.length === 0) return
    state.undoStack.push(state.dslCode)
    const next = state.redoStack.pop()!
    parseAndRender(next)
  }

  return {
    state,
    parseAndRender,
    syncCodeFromAST,
    addCommand,
    modifyCommand,
    deleteCommand,
    pushSnapshot,
    undo,
    redo,
  }
}

// 单例导出
export const editorStore = createEditorStore()
----

=== 数据流时序（4 种核心场景）

==== 场景 A：用户在代码编辑器中输入

[source]
----
用户输入 → CodeMirror 6 updateListener
  → emit('update:value', newCode)
  → editorStore.parseAndRender(newCode)
    → parseDSL(newCode) → { program, errors }
    → state.program = program
    → state.errors = errors
  → CanvasEditor watch('program') fires
    → 清空叠加层
    → render(program) → ImageData
    → putImageData 到 renderCanvas
    → TemplateOverlay 重绘
  → StatusBar watch('errors') fires → 更新状态
  → ErrorPanel watch('errors') fires → 显示/隐藏错误列表
----

==== 场景 B：用户在画布绘制矩形

[source]
----
CanvasEditor: mousedown（记录起点）
  → 当前工具 RectTool.onPointerDown({ x, y })
    → state.toolPreview = { start, end }（预览矩形）
  → ToolPreviewOverlay 响应式重绘（橡皮筋效果）

CanvasEditor: mousemove
  → RectTool.onPointerMove({ x, y })
    → state.toolPreview = { start, end }

CanvasEditor: mouseup（终点确定）
  → RectTool.onPointerUp({ x, y })
    → 返回 ToolResult { type: 'add', command: RectCmd }
  → editorStore.addCommand(rectCmd)
    → pushSnapshot() ← 记录撤销快照
    → program.commands.push(rectCmd)
    → generateDSL(program) → newCode
    → state.dslCode = newCode
  → CanvasRenderer watch('program') → 重绘
  → CodeEditor watch('dslCode') → 更新编辑器内容
  → state.toolPreview = null → 清除预览
----

==== 场景 C：用户选中图形（画布 → 代码联动）

[source]
----
CanvasEditor: click（画布空白区单击）
  → hitTest(x, y, commands) → 返回命令索引
  → editorStore.state.selectedCommandIndices = [index]

== generateDSL() 函数 — 核心新增

=== 接口与实现

[source,typescript]
----
// src/core/index.ts 中新增

/**
 * 将 DSLProgram AST 重新生成为 DSL 源代码字符串
 *
 * 与 parseDSL 严格互逆：
 *   parseDSL(generateDSL(program)) 应得到结构等价的 AST
 *
 * @param program - DSL AST
 * @param options - 生成选项
 * @returns DSL 源代码（末尾带换行）
 */
export function generateDSL(
  program: DSLProgram
): string {
  const lines: string[] = []

  // ----- head -----
  lines.push(`dsl ${program.header.version}`)
  lines.push(`size ${program.header.width} ${program.header.height}`)
  if (program.header.template !== 'none') {
    lines.push(`template ${program.header.template}`)
  }
  lines.push('')

  // ----- palette -----
  lines.push('palette {')
  for (const [code, hex] of program.palette) {
    lines.push(`  ${code} ${hex}`)
  }
  lines.push('}')
  lines.push('')

  // ----- series -----
  for (const s of program.series) {
    lines.push(`series ${s.colors.join(' ')}`)
  }
  if (program.series.length > 0) lines.push('')

  // ----- defines -----
  for (const [, def] of program.defines) {
    lines.push(`define ${def.name} {`)
    for (const cmd of def.commands) {
      lines.push(`  ${drawCommandToString(cmd)}`)
    }
    lines.push('}')
    lines.push('')
  }

  // ----- commands -----
  for (const cmd of program.commands) {
    if (cmd.type === 'use') {
      const transforms = cmd.transforms.length > 0
        ? ' ' + cmd.transforms.join(' ')
        : ''
      lines.push(
        `use ${cmd.name}${transforms} ${cmd.position.x},${cmd.position.y}`
      )
    } else {
      lines.push(drawCommandToString(cmd))
    }
  }

  return lines.join('\n') + '\n'
}

function drawCommandToString(cmd: DrawCommand): string {
  switch (cmd.type) {
    case 'rect':
      return `rect ${cmd.p1.x},${cmd.p1.y} ${cmd.p2.x},${cmd.p2.y} ${cmd.color}`
    case 'line':
      return `line ${cmd.p1.x},${cmd.p1.y} ${cmd.p2.x},${cmd.p2.y} ${cmd.radius} ${cmd.color}`
    case 'poly':
      return `poly ${cmd.points.map(p => `${p.x},${p.y}`).join(' ')} ${cmd.color}`
    case 'bitmap':
      return `bitmap ${cmd.origin.x},${cmd.origin.y} {\n  ${cmd.rows.join('\n  ')}\n}`
  }
}

=== 图形命中检测

[source,typescript]

/**
 * 从后往前遍历命令，判断鼠标点击属于哪个图形
 */
function hitTest(
  dslX: number,
  dslY: number,
  commands: (DrawCommand | UseCmd)[]
): number {
  for (let i = commands.length - 1; i >= 0; i--) {
    const cmd = commands[i]
    if (cmd.type === 'use') continue  // 二期支持 use 展开

    switch (cmd.type) {
      case 'rect': {
        const x1 = Math.min(cmd.p1.x, cmd.p2.x)
        const y1 = Math.min(cmd.p1.y, cmd.p2.y)
        const x2 = Math.max(cmd.p1.x, cmd.p2.x)
        const y2 = Math.max(cmd.p1.y, cmd.p2.y)
        if (dslX >= x1 && dslX <= x2 && dslY >= y1 && dslY <= y2) return i
        break
      }
      case 'line': {
        const dist = pointToSegmentDist(dslX, dslY, cmd.p1, cmd.p2)
        if (dist <= cmd.radius + 1) return i
        break
      }
      case 'poly': {
        if (pointInPolygon(dslX, dslY, cmd.points)) return i
        break
      }
      case 'bitmap': {
        const col = dslX - cmd.origin.x
        const row = dslY - cmd.origin.y
        if (
          row >= 0 && row < cmd.rows.length &&
          col >= 0 && col < cmd.rows[row].length &&
          cmd.rows[row][col] !== '0'
        ) return i
        break
      }
    }
  }
  return -1
}

class SelectTool extends BaseTool {
  name = 'select'
  private dragStart: Point | null = null
  private dragCommandIndex: number = -1
  private dragOriginalCommand: DrawCommand | null = null

  onPointerDown(event: ToolEvent, ctx: ToolContext) {
    const index = hitTest(event.x, event.y, ctx.program?.commands ?? [])
    if (index >= 0) {
      // 命中图形 → 进入拖拽模式
      this.dragStart = { x: event.x, y: event.y }
      this.dragCommandIndex = index
      this.dragOriginalCommand = { ...ctx.program!.commands[index] } as DrawCommand
      return { type: 'select', selectedIndices: [index] }
    }
    // 未命中 → 取消选中
    this.reset()
    return { type: 'select', selectedIndices: [] }
  }

  onPointerMove(event: ToolEvent, ctx: ToolContext) {
    if (!this.dragStart) return null
    // 返回预览数据（CanvasEditor 用来绘制移动中预览）
    const offsetX = event.x - this.dragStart.x
    const offsetY = event.y - this.dragStart.y
    return { type: 'none', preview: { dragCommandIndex: this.dragCommandIndex, offsetX, offsetY } }
  }

  onPointerUp(event: ToolEvent, ctx: ToolContext) {
    if (!this.dragStart || this.dragCommandIndex < 0) return null
    const offsetX = event.x - this.dragStart.x
    const offsetY = event.y - this.dragStart.y
    if (offsetX === 0 && offsetY === 0) {
      this.reset()
      return null  // 没有拖拽 → 仅选中
    }
    // 修改命令坐标
    const cmd = this.dragOriginalCommand!
    const newCmd = offsetCommand(cmd, offsetX, offsetY)
    this.reset()
    return { type: 'modify', commandIndex: this.dragCommandIndex, command: newCmd }
  }

  onKeyDown(key: string, ctx: ToolContext) {
    if (key === 'Delete' || key === 'Backspace') {
      // 删除选中图形由 editorStore 处理，这里返回提示
      return { type: 'delete', commandIndex: ctx.program?.commands.length ?? -1 }
    }
    return null
  }

  reset() {
    this.dragStart = null
    this.dragCommandIndex = -1
    this.dragOriginalCommand = null
  }
}

 *   idle → drawing (mousedown 第一个点)

class BitmapTool extends BaseTool {
  name = 'bitmap'
  private paintedPixels: Set<string> = new Set()  // "x,y" → color
  private origin: Point | null = null
  private isErasing = false

  onPointerDown(event: ToolEvent, ctx: ToolContext) {
    this.isErasing = event.button === 2
    this.origin = { x: event.x, y: event.y }
    this.paintedPixels.add(`${event.x},${event.y}`)
    return { type: 'none', preview: { pixels: [...this.paintedPixels], color: this.isErasing ? '0' : ctx.currentColor } }
  }

  onPointerMove(event: ToolEvent, ctx: ToolContext) {
    if (!this.origin) return null
    this.paintedPixels.add(`${event.x},${event.y}`)
    return { type: 'none', preview: { pixels: [...this.paintedPixels], color: this.isErasing ? '0' : ctx.currentColor } }
  }

  onPointerUp(event: ToolEvent, ctx: ToolContext) {
    if (!this.origin) return null
    const bitmap = this.pixelsToBitmap(this.paintedPixels, this.origin, ctx.currentColor)
    this.reset()
    return { type: 'add', command: bitmap }
  }

  private pixelsToBitmap(pixels: Set<string>, origin: Point, color: string): BitmapCmd {
    // 计算包围盒
    let minX = Infinity, minY = Infinity, maxX = -Infinity, maxY = -Infinity
    for (const key of pixels) {
      const [x, y] = key.split(',').map(Number)
      minX = Math.min(minX, x)
      minY = Math.min(minY, y)
      maxX = Math.max(maxX, x)
      maxY = Math.max(maxY, y)
    }
    const cols = maxX - minX + 1
    const rows = maxY - minY + 1
    const grid: string[][] = Array.from({ length: rows }, () => Array(cols).fill('0'))
    for (const key of pixels) {
      const [x, y] = key.split(',').map(Number)
      grid[y - minY][x - minX] = color
    }
    return {
      type: 'bitmap',
      origin: { x: minX, y: minY },
      rows: grid.map(row => row.join('')),
    }
  }

  reset() {
    this.paintedPixels.clear()
    this.origin = null
  }
}

      if (!this.view) return
      const current = this.view.state.doc.toString()
      if (code !== current) {
        this.view.dispatch({
          changes: { from: 0, to: current.length, insert: code },
        })
      }
    },
    updateHighlights(lines: number[]) {
      // 设置行高亮 decoration
      if (lines.length > 0 && this.view) {
        const line = this.view.state.doc.line(lines[0])
        this.view.dispatch({
          effects: EditorView.scrollIntoView(line.from, { y: 'center' }),
        })
      }
    },
  },

  watch: {
    value(newVal) {
      if (!this.isInternalChange) {
        this.setEditorValue(newVal)
      }
    },
    highlightLines(lines) {
      this.updateHighlights(lines)
    },
  },

  mounted() {
    const startState = EditorState.create({
      doc: this.value,
      extensions: this.extensions,
    })
    this.view = new EditorView({
      state: startState,
      parent: this.$refs.editorContainer as HTMLElement,
    })
  },

  beforeUnmount() {
    this.view?.destroy()
  },
})
</script>

// CodeEditor 组件内部：dsl-language.ts
import { StreamLanguage } from '@codemirror/language'

export const dslLanguage = () => StreamLanguage.define({
  token(stream) {
    // 注释
    if (stream.match(/^\/\/.*/)) return 'comment'

    // 空白
    if (stream.eatSpace()) return null

    // DSL 关键字（命令 + 声明）
    if (stream.match(
      /^(dsl|size|template|palette|series|define|use|rect|line|poly|bitmap)\b/
    )) return 'keyword'

    // 变换关键字
    if (stream.match(/^(fx|fy|r90|r180|r270)\b/)) return 'keyword'

    // 十六进制颜色 #RRGGBB
    if (stream.match(/^#[0-9a-fA-F]{6}/)) return 'color'

    // 数字（整数）
    if (stream.match(/^-?\d+/)) return 'number'

    // 单字符颜色代码（1-9, a-z, A-Z）
    if (stream.match(/^[0-9a-zA-Z]/)) return 'atom'

    // 花括号 / 逗号
    if (stream.match(/^[{}|,]/)) return 'punctuation'

    // 其他
    stream.next()
    return null
  },
  languageData: {
    commentTokens: { line: '//' },
  },
})

// stores/useTemplateRegion.ts（独立工具函数，非 composable）
export interface RegionInfo {
  isDrawable: (x: number, y: number) => boolean
  drawableBoundary: Point[]   // 可编辑区域边界（用于绘制边界线）
  maskRegions: MaskRegion[]   // 需要遮罩的区域
}

export interface MaskRegion {
  type: 'rect' | 'polygon'
  points: Point[]   // 遮罩区域顶点
}

export function getTemplateRegion(header: DSLHeader): RegionInfo {
  const halfW = Math.floor(header.width / 2)
  const halfH = Math.floor(header.height / 2)

  switch (header.template) {
    case 'none':
      return {
        isDrawable: () => true,
        drawableBoundary: [],
        maskRegions: [],
      }

    case 'unit':
    case 'turret':
      return {
        isDrawable: (x) => x < halfW,
        drawableBoundary: [
          { x: halfW, y: 0 },
          { x: halfW, y: header.height - 1 },
        ],
        maskRegions: [{
          type: 'rect',
          points: [
            { x: halfW, y: 0 },
            { x: header.width, y: 0 },
            { x: header.width, y: header.height },
            { x: halfW, y: header.height },
          ],
        }],
      }

    case 'block':
      return {
        isDrawable: (x, y) => y < halfH && x + y < header.width,
        drawableBoundary: [
          { x: 0, y: 0 },
          { x: header.width - 1, y: 0 },
          { x: halfW, y: halfH },
        ],
        maskRegions: [{
          // 三角形外部 = 整个矩形减去三角形
          type: 'polygon',
          points: [
            { x: 0, y: 0 },
            { x: header.width, y: 0 },
            { x: header.width, y: header.height },
            { x: 0, y: header.height },
            { x: 0, y: 0 },
            { x: halfW, y: halfH },
            { x: header.width, y: 0 },
          ],
        }],
      }
  }
}

<script lang="ts">
import { defineComponent } from 'vue'
import { getTemplateRegion } from '../../stores/useTemplateRegion'

export default defineComponent({
  name: 'TemplateOverlay',
  props: {
    header: { type: Object, required: true },
    scale: { type: Number, required: true },
  },
  inject: ['editorStore'],

  computed: {
    canvasWidth() {
      return this.header.width * this.scale
    },
    canvasHeight() {
      return this.header.height * this.scale
    },
    regionInfo() {
      return getTemplateRegion(this.header)
    },
  },

  watch: {
    header: { handler: 'draw', deep: true },
    scale: 'draw',
  },

  methods: {
    draw() {
      const canvas = this.$refs.overlayCanvas as HTMLCanvasElement
      const ctx = canvas.getContext('2d')
      if (!ctx) return
      const scale = this.scale

      ctx.clearRect(0, 0, canvas.width, canvas.height)

      // 1. 绘制半透明遮罩
      ctx.fillStyle = 'rgba(0, 0, 0, 0.4)'
      for (const region of this.regionInfo.maskRegions) {
        ctx.beginPath()
        ctx.moveTo(region.points[0].x * scale, region.points[0].y * scale)
        for (let i = 1; i < region.points.length; i++) {
          ctx.lineTo(region.points[i].x * scale, region.points[i].y * scale)
        }
        ctx.closePath()
        ctx.fill()
      }

      // 2. 绘制可编辑区域边界（白色虚线）
      if (this.regionInfo.drawableBoundary.length > 0) {
        ctx.strokeStyle = '#ffffff'
        ctx.lineWidth = 2
        ctx.setLineDash([6, 4])
        ctx.beginPath()
        ctx.moveTo(
          this.regionInfo.drawableBoundary[0].x * scale,
          this.regionInfo.drawableBoundary[0].y * scale
        )
        for (let i = 1; i < this.regionInfo.drawableBoundary.length; i++) {
          ctx.lineTo(
            this.regionInfo.drawableBoundary[i].x * scale,
            this.regionInfo.drawableBoundary[i].y * scale
          )
        }
        ctx.stroke()
        ctx.setLineDash([])
      }

      // 3. 添加提示文字
      if (this.regionInfo.drawableBoundary.length > 0) {
        ctx.fillStyle = 'rgba(255, 255, 255, 0.7)'
        ctx.font = `${12 * Math.min(1, scale / 4)}px sans-serif`
        if (this.header.template === 'block') {
          ctx.fillText('★ 可编辑区域（三角形）', 4, 14)
        } else {
          ctx.fillText('★ 可编辑区域（左侧）', 4, 14)
        }
      }
    },
  },

  mounted() { this.draw() },
})
</script>

== 撤销/重做机制

=== 设计原则

- 基于 **DSL 文本快照**（而非 AST 快照）
- 快照在每次"用户操作完成"时记录
- 最大保留 100 步历史

=== 触发时机

|====
| 操作 | 触发时机 | 记录内容
| 文本编辑（CodeMirror）| debounce 300ms 后 | 当前 DSL 代码
| 矩形绘制完成 | mouseup | 添加 rect 命令后的 DSL 代码
| 线条绘制完成 | mouseup | 添加 line 命令后的 DSL 代码
| 多边形闭合 | 双击/右键 | 添加 poly 命令后的 DSL 代码
| 画笔绘制完成 | mouseup | 添加 bitmap 命令后的 DSL 代码
| 图形拖拽移动 | mouseup | 坐标修改后的 DSL 代码
| 删除图形 | Delete 键 | 删除后的 DSL 代码
|====

=== 实现

撤销/重做已在 [editorStore](#editorstore-全局状态设计) 中实现，核心逻辑：

[source,typescript]
----
function pushSnapshot(): void {
  const current = state.dslCode
  // 避免重复记录相同状态
  if (state.undoStack.length > 0 && state.undoStack[state.undoStack.length - 1] === current) return
  state.undoStack.push(current)
  if (state.undoStack.length > 100) state.undoStack.shift()
  state.redoStack = []  // 新操作清空重做栈
}

function undo(): void {
  if (state.undoStack.length === 0) return
  state.redoStack.push(state.dslCode)
  const prev = state.undoStack.pop()!
  parseAndRender(prev)
}

function redo(): void {
  if (state.redoStack.length === 0) return
  state.undoStack.push(state.dslCode)
  const next = state.redoStack.pop()!
  parseAndRender(next)
}

文本快照方案 vs AST 快照方案：

[cols="1,2,2"]
|===
| 维度 | 文本快照（选用） | AST 快照（放弃）
| 内存占用 | 每个快照为字符串，128×128 文件约 1-2KB，100 步约 200KB | 每个快照为 AST 对象，可能达数十 KB
| 实现复杂度 | 极低：push/pop 字符串 | 需要深拷贝 DSLProgram（含 Map、嵌套对象）
| 互逆性 | `parseDSL(generateDSL(program))` 保证 | 直接操作 AST，需保证一致性
| 性能 | 仅记录文本，parse 在恢复时执行一次 | 需深拷贝，但恢复时不需 re-parse
|===

// composables/useKeyboard.ts
export function useKeyboard(editorStore: any) {
  function onKeyDown(event: KeyboardEvent) {
    // 如果焦点在 CodeMirror 中，不处理全局快捷键（除非 Ctrl+）
    if (isCodeMirrorFocused() && !event.ctrlKey && !event.metaKey) return

    const toolMap: Record<string, ToolType> = {
      'v': 'select', 'V': 'select',
      'r': 'rect',   'R': 'rect',
      'l': 'line',   'L': 'line',
      'p': 'poly',   'P': 'poly',
      'b': 'bitmap', 'B': 'bitmap',
      'i': 'eyedropper', 'I': 'eyedropper',
    }

    if (toolMap[event.key]) {
      editorStore.state.currentTool = toolMap[event.key]
      event.preventDefault()
      return
    }

    if (event.key === 'Escape') {
      editorStore.state.selectedCommandIndices = []
      // 通知当前工具 reset
      event.preventDefault()
    }
  }

  return { onKeyDown }
}

== 工程风险与应对

[cols="1,2,3"]
|===
| 风险 | 场景 | 应对方案
| 频繁 parse/render 卡顿 | 用户快速输入 DSL 文本 | debounce 150ms；CodeMirror 的 updateListener 本身是增量更新
| 拖拽中全量 generateDSL | 拖拽移动图形 | 拖拽中仅更新 canvas（预览层），mouseup 后再同步到代码
| 大画布渲染慢 | 256×256 以上大图 | render() 本身 O(N) 复杂度（N=像素数），256×256=65536 像素，实测 <5ms；超过 512×512 考虑 Web Worker
| bitmap 大块编辑性能 | 画笔模式绘制大面积 | 限制 bitmap 包围盒最大值（建议 ≤64×64）；CodeMirror 6 虚拟滚动不影响
| Vue 响应式深层对象 | Map 类型（palette/defines）不是 reactive 的 | 直接替换整个 Map；不做深层响应
| 代码编辑器和画布相互更新循环 | 文本编辑 → AST → 代码同步 → 文本编辑 | 通过 isInternalChange 标志位防止循环
| build-web.js 废弃后 CI 需更新 | 构建脚本 | `npm run build` 直接调 `vite build`，废弃的 build-web.js 保留但不再使用
|===
